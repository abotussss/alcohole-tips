# Alcohol Atlas

日本酒とワインを中心に、品種・ブランド・製法・味わいの違いをビジュアルで比較できる Next.js 製の Web アプリです。

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Features

- モダンでシンプルなトップページ
- 日本酒カテゴリ
  - ブランド軸
  - 製法、製造会社、都道府県、味わいの比較
- ワインカテゴリ
  - 品種軸
  - 主要産地と味わいの比較
- 5角形レーダーチャート
- 実写の代わりに抽象イラストで構成

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
